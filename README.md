## 📌 About this project

This desktop application manages student attendance through face recognition. It removes the need for manual roll calls. The system identifies students as they enter the classroom. It records their presence automatically. This application works entirely offline, which keeps student data on your local device. The software uses advanced computer vision to ensure high accuracy. It serves teachers and administrators who need a reliable way to track attendance.

## 🛠️ Key features

- Automatic face recognition for fast check-ins.
- Manual override mode for attendance adjustments.
- Department filtering for organized records.
- CSV export for integration with other systems.
- One-click backup tools for data security.
- Offline storage to protect user privacy.

## 🚀 Setting up the application

Follow these steps to prepare the system for use:

1. **Launch the software:** Locate the shortcut on your desktop and open it.
2. **Setup your database:** On the first run, the app prompts you to create a storage folder for your student records. Pick a secure location on your hard drive.
3. **Register students:** Navigate to the "Enrollment" tab.
4. **Input details:** Enter the student name, ID number, and department.
5. **Capture faces:** Select the "Capture" button. Point the camera at the student. The system detects the face and saves the data. Repeat this for every student.
6. **Save changes:** Select the "Save" button to commit the profiles to the system.

## 📑 Managing attendance

Once you finish registration, you can start tracking students.

1. **Start session:** Select the "Live" tab in the main menu.
2. **Choose mode:** Use "Auto" for automatic recognition or "Manual" if you need to select a name from a list.
3. **Monitor stream:** A window displays the camera view. A green box appears around faces recognized by the system.
4. **View logs:** The system logs entry times immediately. You can view these logs in the "History" tab.

## 📤 Exporting your records

You might need to send attendance records to a department head or save them for your own files.

1. Open the "Reports" tab in the application.
2. Select your date range.
3. Select the "Export to CSV" button.
4. Choose the folder where you want to save the file.
5. Open the folder to confirm your file exists. You can open this file with Microsoft Excel or any spreadsheet tool.

## 💾 Backing up data

Data loss causes major headaches. Always keep a backup.

1. Navigate to the "Settings" menu.
2. Find the section labeled "Data Management."
3. Select the "Create Backup" button.
4. Select a safe location, such as a USB flash drive or a cloud storage folder.
5. Store this backup in a separate location from your main computer.

## ❓ Troubleshooting common issues

If you encounter problems, consult these solutions:

- **Camera not detected:** Ensure no other app uses the camera. Check your Windows privacy settings to grant the app camera access.
- **Low accuracy:** Make sure students face the camera directly. Improve the lighting in the room if the system struggles to identify faces.
- **App freezes:** Wait a few moments for the system to process the data. If it remains unresponsive, restart the software.
- **Missing students:** Check if you enrolled the student correctly in the "Enrollment" tab. Ensure the face data shows in the profile preview.
